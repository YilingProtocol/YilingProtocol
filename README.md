# Yiling Protocol — Live on Base

Oracle-free, self-resolving prediction markets powered by game theory. Built on Harvard's SKC mechanism.

## Overview

Yiling Protocol is a truth discovery infrastructure that eliminates oracle dependency. Markets resolve themselves through mathematically proven game-theoretic incentives — honest reporting is always the dominant strategy.

## Live

- **Network:** Base
- **Markets:** [Explore Live Markets](https://yilingmarket-onbase.vercel.app/markets)
- **Docs:** [Documentation](https://yiling-protocol-landing.vercel.app/docs/getting-started/overview)

## Tech Stack

- **Smart Contracts:** Solidity (MarketFactory, PredictionMarket, FixedPointMath)
- **Frontend:** Next.js, Tailwind CSS, Framer Motion
- **Mechanism:** SKC (Srinivasan-Karger-Chen) with cross-entropy scoring

## Research

Based on peer-reviewed research from Harvard, published at ACM EC 2025.

- [Self-Resolving Prediction Markets — arXiv](https://arxiv.org/abs/2306.04305)
